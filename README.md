# <Prework Study Guide Webpage>

## A brief intro into GitHub, HTML, CSS, and JavaScript.

The motivation was to get an introduction to Github, HTML, CSS, and Javascript. Through this introduction we created a study guide that is easily navigated. It solves the problem of learning a lot of material and structuring something that makes a good reference for the rest of the coding bootcamp class. I learned basics about coding and how to integrate it with GitHub which will be used throughout the remainder of the course.

This Study Guide can be used to reference material learned in the prework assignment. There are notes pertaining to Github, HTML, CSS, and JavaScipt.
